# OptiCrop: Project Demo Planning

## Demo Flow
1. Introduce the problem: helping farmers choose the right crop for their soil and climate.
2. Show the input form and enter sample soil/climate values.
3. Submit and show the predicted crop result.
4. Briefly walk through the notebook: EDA -> preprocessing -> model training -> evaluation -> K-Means clustering.
5. Summarize results: 97.27% accuracy, ROC-AUC 0.9998, and the seasonal/cluster insights.
